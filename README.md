## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript, TypeORM, PostgreSQL starter repository.

## Installation

```bash
$ npm install
```

## Running the app

1 Create own .env config:

```bash
cp .env.example .env
```

2 Generate typeorm's  `ormconfig.json` file:

```bash
npm run typeormconfig
```

3 Run the application

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
